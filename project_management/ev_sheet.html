1. Earned Value Management Methodology Framework

Earned Value Management (EVM) is a project management methodology used to measure project performance and progress by integrating scope, schedule, and cost parameters.

Core Parameters (Cumulative Values)

  • PV – Planned Value: The authorized budget assigned to the work scheduled to be accomplished for an activity.
  • AC – Actual Cost: The total cost actually incurred and recorded in accomplishing the work performed for an activity.
  • EV – Earned Value: The value of work performed expressed in terms of the approved budget assigned to that work for an activity.

2. Earned Value Indicators & Mathematical Formulas Formulas

  • Schedule Variance (SV): A measure of schedule performance calculated by:
    SV = EV − PV
  • Cost Variance (CV): A measure of cost performance calculated by:
    CV = EV − AC
  • Schedule Performance Index (SPI): A measure of progress achieved compared to progress planned on a project:
    SPI = EV / PV
  • Cost Performance Index (CPI): A measure of the value of work completed compared to the actual cost incurred on the project:
    CPI = EV / AC
