5. Post-Implementation Lessons Learned & Retrospective

Technical & Engineering Insight

• File System Concurrency: Relying on JSON file storage in Express requires explicit mutex or queue locks to handle asynchronous non-blocking I/O. Future projects should migrate to SQLite or PostgreSQL for multi-worker scaling.
• Docker Non-Root Volume Mounts: Containerizing applications under security-hardened Alpine Linux images requires setting explicit owner permissions on data directories before application boot.

AI Agent Governance & Prompt Engineering

• Tool Call Boundaries: Large Language Models (e.g. Gemini 2.5/3.6) require strictly typed JSON function signatures and boundary constraints in the system prompt to prevent argument hallucination or broken parameter types during multi-turn chats.
